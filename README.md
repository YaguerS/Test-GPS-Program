# Android GPS Tracking Project

## 📱 Overview

This project is an Android application developed as a technical learning exercise focused on GPS location tracking, mobile application structure, and basic Android development concepts.

The goal is to understand how location services work in Android and practice building a structured mobile app.

---

## 🎯 Purpose

- Learn Android development fundamentals
- Work with GPS/location APIs
- Understand Android lifecycle
- Build a foundation for mobile applications

---

## ⚙️ Features

### Current
- Basic Android project structure
- GPS location access testing
- Prototype location retrieval

### Planned
- Real-time tracking UI
- Background location service
- Data logging
- Export/sync functionality

## 🧠 Learning Outcomes

- App lifecycle understanding
- Debugging mobile apps
